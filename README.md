## `$ ls projects/`

### 🔒 Stealth-mode SaaS *(in progress)*
Cloud-native B2C/B2B platform built from scratch — event-driven, geospatially-aware, privacy-first.  
Stack: FastAPI · Next.js · PostGIS · Celery · Redis · AWS. More details when it ships.

### 🌐 [zapf.dev](https://zapf.dev)
Personal site, technical blog, and **public atlas** — a knowledge graph of atomic cybersecurity concepts (definitions, prereqs, retrieval prompts, references) interconnected with the long-form write-ups on the blog. Built with **Astro 6** and **Catppuccin**: a unified D3.js graph (4 node groups, context-aware highlight per page), Pagefind full-text search (⌘K), Mermaid diagrams, lazy-loaded Wavesurfer.js audio player, dynamic OG images (Satori + Sharp), JSON-LD structured data, Giscus comments, privacy-friendly Umami analytics.  
→ [live](https://zapf.dev) · [atlas](https://zapf.dev/atlas) · [graph](https://zapf.dev/graph) · [source](https://github.com/MatiasZapf/zapf.dev)

### 🧬 atomic-brain *(personal R&D)*
Multi-agent automation pipeline (Claude Code) that ingests heterogeneous lecture material — PDFs, video transcripts, reference textbooks — and produces auditable, self-contained study notes for my Cyberdefense coursework. DAG of specialised agents · retrieval over indexed sources · deterministic quality gates · idempotent re-runs · per-step model routing. Pedagogical foundation: Ausubel · Sweller · Mayer · Bloom · Matuschak.  
A subset of the atomic concepts surfaces publicly at [zapf.dev/atlas](https://zapf.dev/atlas).

### ⚙️ [dotfiles](https://github.com/MatiasZapf/dotfiles)
Fedora 44 / COSMIC desktop, managed with GNU Stow. Catppuccin Frappé end-to-end.  
→ Alacritty · tmux · LazyVim · Starship · ble.sh + fzf
