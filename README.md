## Independent Projects & Public Work

### [apuntes.md](https://apuntes.md) — pedagogical notes platform for my Cyberdefense degree *(private)*
A self-contained notebook for my B.S. in Cyberdefense: each class rebuilt from scratch — theory, solved practice (lecturer-mode then expert-mode), and active-recall self-assessment — with load-bearing concepts extracted into reusable atomic notes linked across the whole degree. Built on Quartz, deployed on Cloudflare, and produced by a multi-agent pipeline (DAG orchestration, auditor↔enricher auto-loops, deterministic gates, auto-generated TikZ figures) grounded in evidence-based pedagogy (Ausubel · Sweller · Mayer · Bloom · Matuschak).

### [Atlas](https://zapf.dev/atlas) & [technical blog](https://zapf.dev/blog) — public knowledge graph & technical writing
A public atlas of cybersecurity concepts as atomic, interlinked nodes — definitions, prerequisites, common misconceptions, and retrieval prompts — taking the same agentic note pipeline behind **apuntes.md** (above) into public. Nodes are seeded from my long-form blog write-ups on cybersecurity, learning systems, and developer tooling, which connect hands-on labs and academic material back to these atomic concepts.
